Памятка по DevTools (шпаргалка)
Открыть и выбрать инструмент

Открыть DevTools: F12 или Ctrl+Shift+I

Включить режим мобильного просмотра: в DevTools нажать иконку смартфона/планшета (или Ctrl+Shift+M)

Вкладки и что в них делать

Elements (Просмотр кода)

Клик правой кнопкой по любому элементу на сайте → «Посмотреть код» (Inspect).

Слева HTML, справа CSS (стили). Можно смотреть цвет, шрифт, отступы.

Двойной клик по свойству в CSS — можно временно изменить значение и увидеть результат.

Console (Консоль)

Показывает ошибки (красные строки), предупреждения, сообщения.

Можно вводить свои команды JavaScript, но для начала просто читай ошибки.

После обновления страницы (F5) смотри, какие ошибки появляются.

Network (Сеть)

Показывает все запросы, которые отправляет страница.

Чтобы увидеть запросы: открой вкладку, потом выполни действие (например, вход).

Кликни по запросу → вкладка Payload покажет отправленные данные (логин, пароль).

Вкладка Response покажет ответ сервера.

Поиск по элементам

Находясь во вкладке Elements, нажми Ctrl+F — откроется поисковая строка, можно искать текст, теги, классы.
